# OSM Localizer


**OSM Localizer** is a web-based OpenStreetMap tool designed to simplify and streamline localization of map features. It lets contributors work on structured challenges where OSM elements from a selected area are assigned for translation and tagging in different languages.

Challenge creators define areas using a map interface and Overpass queries. Contributors then receive random elements and can edit names/tags directly in the tool before submitting updates back to OpenStreetMap.

## Key Features

- OSM OAuth2 authentication via OpenStreetMap  
- Challenge creation using map-drawn areas + Overpass queries  
- Random OSM element assignment for localization tasks  
- Built-in editor for multilingual tagging  
- Direct upload of edits to OpenStreetMap  
- Optional translation engine integration  

## Getting Started

Log in using your OpenStreetMap account, select a challenge, and start localizing assigned elements. Each task provides an OSM feature that can be edited and saved directly in the platform. Once completed, changes can be uploaded back to OpenStreetMap.
